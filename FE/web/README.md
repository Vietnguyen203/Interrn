# Frontend Web (FE Web)

Ứng dụng web dành cho Admin và Quầy thu ngân, được xây dựng bằng React + Vite.

## Cấu trúc công nghệ
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Vanilla CSS / Tailwind (tùy cấu hình)
- **State Management**: React Hooks
- **Communication**: REST API (qua Gateway 8080) & WebSocket (StompJS)

## Cách chạy
1. Cài đặt Node.js (phiên bản 18 trở lên).
2. Chạy lệnh cài đặt thư viện:
   ```bash
   npm install
   ```
3. Chạy ứng dụng:
   ```bash
   npm run dev
   ```
4. Mở trình duyệt: `http://localhost:5173`
